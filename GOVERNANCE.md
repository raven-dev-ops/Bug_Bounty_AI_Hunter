# Governance

This project is maintained by Raven Development Operations.

## Decision making
- Maintainers make decisions by lazy consensus.
- If consensus cannot be reached, the primary maintainer decides.

## Changes
- Significant content or policy changes should be proposed in an issue.
- Minor fixes can be proposed directly via pull request.

## Conduct
All participants must follow `CODE_OF_CONDUCT.md`.

## Contact
For governance questions, email support@ravdevops.com.
