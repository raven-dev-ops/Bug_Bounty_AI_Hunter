# Knowledge Tags

Use these tags consistently across cards and checklists. Add new tags only
when needed and document them here.

## AI surfaces
- `ai-security`: AI system security
- `rag`: retrieval augmented generation
- `embeddings`: embeddings and vector stores
- `fine-tuning`: model fine-tuning and training data
- `logging`: prompt and telemetry logging
- `prompt-injection`: prompt manipulation and leakage
- `agents`: tool-using or autonomous agents

## Application security
- `api-security`: API authentication and request integrity
- `access-control`: authorization and least privilege
- `privacy`: sensitive data handling
- `telemetry`: telemetry and monitoring systems

## Platforms and environments
- `mobile`: mobile clients and APIs
- `windows`: Windows tooling
- `legacy-systems`: mainframe and legacy systems
- `endpoint-security`: endpoint security frameworks

## Process and operations
- `human-factors`: human-in-the-loop considerations
- `supply-chain`: update and dependency pipelines
- `reverse-engineering`: client and protocol analysis
