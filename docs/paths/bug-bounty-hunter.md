# Path: Bug Bounty Hunter

This path focuses on running the workflow end to end for authorized targets.

## Steps
1. Read `ROE.md` and confirm written authorization.
2. Build a TargetProfile via `TARGET_PROFILE.md`.
3. Generate a threat model and dataflow map (`THREAT_MODEL.md`).
4. Plan the pipeline (`PIPELINE.md`) and review the scan plan.
5. Capture evidence and produce reports (`REPORTING.md`).

## Helpful references
- `HACK_TYPES.md` for coverage ideas.
- `NOTIFICATIONS.md` for alerts and handoffs.
