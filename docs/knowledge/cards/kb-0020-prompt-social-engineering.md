# Prompt social engineering risk

## Metadata
- ID: kb-0020

- Status: reviewed

## Summary
Prompts can influence human operators as well as systems. Review processes should account for social engineering via prompts.

## Relevance to the project
Supports human-in-the-loop guidance for AI security reviews.

## Safe notes
- Require human review for high-impact actions.
- Document escalation paths when prompts request sensitive actions.
- Train reviewers to recognize manipulative prompt patterns.
- Limit model autonomy and scope.
