# Standard Report Templates

This pack provides industry-standard Markdown templates for reports and
per-finding writeups. These templates are not wired into automation yet; use
them for manual report drafting or as a starting point for custom tooling.

## Placeholders
- `{program_name}`
- `{generated_at}`
- `{prepared_by}`
- `{executive_summary}`
- `{scope_summary}`
- `{methodology}`
- `{findings_overview_table}`
- `{findings_markdown}`
- `{recommendations}`
- `{evidence_summary}`
- `{attachments_manifest}`
- `{reproducibility_pack}`
- `{title}`
- `{id}`
- `{severity}`
- `{severity_model_summary}`
- `{affected_assets}`
- `{summary}`
- `{steps}`
- `{impact}`
- `{remediation}`
- `{evidence_refs}`
- `{references}`
