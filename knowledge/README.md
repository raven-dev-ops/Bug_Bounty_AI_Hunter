# Knowledge Base

This folder holds curated sources, derived cards, and checklists used by the
project. All content must follow the ROE and avoid weaponized details.

## Structure
- `sources/` - raw sources and source notes
- `cards/` - safe, distilled knowledge cards
- `checklists/` - reusable checklists for review modules

## Index
Generate `knowledge/INDEX.md` with:
```bash
python -m scripts.knowledge_index --output knowledge/INDEX.md
```
